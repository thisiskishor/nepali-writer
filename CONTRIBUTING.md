# Contributing

Contributions are welcome — improvements to Nepali language guidance, better examples, additional domain coverage, or fixes based on real-world usage.

## How to Contribute

1. Fork the repo
2. Create a branch: `git checkout -b my-improvement`
3. Make your changes
4. Open a pull request with a clear description of what changed and why

## What's Worth Contributing

- **Better examples** in [`nepali-writer/examples/`](nepali-writer/examples/) — Real translations from your domain (business, technical, academic, literary)
  - See current examples: [Example 1](nepali-writer/examples/example-1-formal-letter.md), [Example 2](nepali-writer/examples/example-2-social-media.md), [Example 3](nepali-writer/examples/example-3-technical-docs.md), etc.
  
- **Domain-specific guides** — New sections for legal, medical, technical, marketing Nepali
- **Improved explanations** in [SKILL.md](nepali-writer/SKILL.md) — Clarifications that would help other writers
- **New trigger phrases** in SKILL.md frontmatter — If you found a phrase that should trigger this skill but doesn't, add it
- **Nepal Academy updates** — If you find our standards outdated or incorrect
- **Regional dialect notes** — Awareness of regional variations (when Nepali Writer expands to Newari, Maithili, etc.)
- **Cultural context improvements** — Better explanations of Nepali customs, honorifics, festivals
- **Translation verification** — Real examples showing where AI commonly fails and how to fix it

## Known Items for Review

Some translations in the examples are flagged for refinement and improvements. If you can improve these, please submit a correction via PR!

### 1. Example 1: Formal Letter — "आमाद" vs "हाम्रो"

**Location:** [Example 1: Formal Letter](nepali-writer/examples/example-1-formal-letter.md)

**Issue:** Line 3 translates "our team" to "आमाद टोलीलाई"

**Status:** ✓ CORRECTED to "हाम्रो टोलीलाई"

**What's fixed:** Changed to the more standard "हाम्रो" (our), which is the more common form in formal correspondence.

---

### 2. Example 3: Technical Docs — Disk Space Terminology Consistency

**Location:** [Example 3: Technical Documentation](nepali-writer/examples/example-3-technical-docs.md)

**Issue:** "Disk space" translates as both "डिस्क स्पेस" and "डिस्क स्थान" — no single standard

**Status:** ⚠️ PARTIAL — Both options documented with guidance provided

**Explanation:** 
- **डिस्क स्थान** = partially translated (recommended for formal Nepali documents)
- **डिस्क स्पेस** = fully transliterated (common in tech-savvy communities)

**Contribution idea:** If you know which term is more standard in Nepal Academy, please submit the clarification with source documentation.

---

### 3. Example 5: Translation Comparison — Raswa/Dirgha Spelling

**Location:** [Example 5: Translation Comparison](nepali-writer/examples/example-5-translation-comparison.md)

**Issue:** "Happy" translates as "खुसी" vs "खुशी" (short vs long ी vowel)

**Status:** ✓ CORRECTED to "खुशी" (Dirgha — long vowel is correct)

**What's fixed:** Updated to use "खुशी लाग्यो" (happiness touched me), where the long ी vowel (Dirgha) is the correct Raswa/Dirgha distinction.

## How to Submit Corrections

1. **Open an issue** describing the translation/grammar issue with:
   - Example location (file path)
   - Current text
   - Suggested correction
   - Source/reasoning (Nepal Academy, native speaker feedback, etc.)

2. **Or submit a PR** with:
   - Corrected example file(s)
   - Explanation of what changed and why
   - Reference to Nepal Academy standards or source
   - Brief summary in commit message

3. **Include these details:**
   - Your name (will be credited in CHANGELOG)
   - Any authority source you reference (Nepal Academy, dictionary, etc.)
   - Context or usage examples supporting your correction

**Thank you for making Nepali writing more accurate!**

## What to Avoid

- Don't add features specific to one person's workflow or one company's style guide
- Don't remove core honorific guidance or structural rules — they exist because violating them causes real errors
- Don't make assumptions about what "sounds better" without grounding in Nepal Academy standards
- Don't add major new sections without discussing scope first

## Standards to Follow

All contributions should:

- Follow **Nepal Academy** standards for orthography, grammar, honorifics
- Include **worked examples** when introducing new concepts
- Use **clear, plain English** in explanations
- Include **before/after examples** when suggesting improvements
- Reference **authoritative sources** (Nepal Academy, Nepali Brihat Shabdakosh, etc.)
- Maintain the **existing structure** and formatting
