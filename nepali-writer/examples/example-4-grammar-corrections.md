# Example 4: Grammar Corrections

Common Nepali writing errors and their corrections.

---

## Error 1: Honorific Mismatch

### ❌ Wrong
```
दिदि यहाँ बस् ।
```

### ✓ Correct (Option 1 - Formal)
```
दिदी यहाँ बस्नुहोस् ।
```

### ✓ Correct (Option 2 - Informal)
```
दिदि यहाँ बस ।
```

### Explanation
- **Error:** Mixing informal pronoun (दिदि) with missing formality marker
- **Fix 1:** Formal: दिदी (with Dirgha) + बस्नुहोस् (formal verb) = consistent honorific level
- **Fix 2:** Informal: दिदि + बस (informal verb) = consistent honorific level
- **Rule:** Pronouns and verbs must match in formality level

## Error 2: Ergative Case with Intransitive Verb + Postposition Spacing

### ❌ Wrong
```
मैले घर मा जान्छु ।
```

### ✓ Correct
```
म घरमा जान्छु ।
```

### Explanation
- **Error 1:** Using ergative case "मैले" with intransitive verb
  - "जानु" (to go) is an intransitive verb
  - Intransitive verbs use nominative case "म", not ergative "मैले"
  - Ergative "मैले" is only for transitive verbs in past tense (e.g., मैले खाना खाएँ — I ate food)
- **Error 2:** Space between noun (घर) and postposition (मा)
- **Fix:** Use nominative "म" (not "मैले") + attach postposition directly (घरमा)
- **Rule:** 
  - Padyog/Padbiyog — postpositions attach directly to nouns without space
  - Case marking: Intransitive verbs require nominative case in all tenses

**Examples of correct attachment and case marking:**
- म घरमा जान्छु। (I go home — nominative "म" + locative postposition -मा)
- उनी स्कुलबाट आई। (She came from school — nominative "उनी" + ablative postposition -बाट)
- हामी काठमाडौँमा रहन्छौं। (We live in Kathmandu — nominative "हामी" + locative -मा)

## Error 3: Incorrect Punctuation

### ❌ Wrong
```
यो राम्रो छ.
```

### ✓ Correct
```
यो राम्रो छ ।
```

### Explanation
- **Error:** Using Latin period (.) instead of Devanagari danda (।)
- **Fix:** Use Devanagari danda (।) for all Nepali sentence endings
- **Rule:** Nepali uses दण्ड (।) not period (.)
- **Why:** Proper script consistency; shows native Nepali writing

**Note:** Double danda (॥) is used for major section breaks.

## Error 4: Compound Verb Splitting

### ❌ Wrong
```
गर्नु हुन्छ ।
```

### ✓ Correct
```
गर्नुहुन्छ ।
```

### Explanation
- **Error:** Splitting the honorific compound verb गर्नुहुन्छ into two words
- **Fix:** Write as one word गर्नुहुन्छ (honorific form)
- **Rule:** Padyog/Padbiyog — honorific compounds are written as one word
- **Why:** गर्नु (to do - base form) + हुन्छ (happens) → गर्नुहुन्छ (formal: "one does")

**Similar examples:**
- बस्नुहोस् (sit - formal) — not: बस्नु होस्
- गर्नुपर्छ (must do - formal) — not: गर्नु पर्छ
- जानुहुन्छ (go - formal) — not: जानु हुन्छ

## Error 5: Raswa/Dirgha Vowel Length

### ❌ Wrong
```
पानि गरेँ ।
```

### ✓ Correct (if meaning "also")
```
पनि गरेँ ।
```

### ✓ Correct (if meaning "water")
```
पानी गरेँ ।
```

### Explanation
- **Error:** Ambiguity between two meanings due to vowel length
- **पनि** (short इ — raswa): means "also" or "too"
- **पानी** (long ी — dirgha): means "water"
- **Context matters:** Choose based on intended meaning
- **Fix:** Use correct vowel length for intended word

**Common Raswa/Dirgha confusions:**
| Word | Raswa (short) | Dirgha (long) | Meaning |
|------|---|---|---|
| दिदि vs दिदी | younger sister | elder sister |
| खुसि vs खुशी | not standardized | happy |
| कुरा vs कुरां | not standardized | matter/thing |

## Error 6: Plural Suffix with Dirgha

### ❌ Wrong
```
मान्छेहरु आए ।
```

### ✓ Correct
```
मान्छेहरू आए ।
```

### Explanation
- **Error:** Plural suffix -हरु written with short उ
- **Fix:** Plural suffix -हरू written with long ू (Dirgha)
- **Rule:** The plural suffix -हरू always has Dirgha (long ू)
- **Why:** Linguistic convention; -हरु is not a standard Nepali morpheme

**Application:**
- मान्छे + हरू = मान्छेहरू (people)
- पुस्तक + हरू = पुस्तकहरू (books)
- बच्चा + हरू = बच्चाहरू (children)

## How to Verify Your Own Writing

Use this checklist when proofreading Nepali text:

- [ ] Pronouns and verbs match in formality level
- [ ] Postpositions are attached (no spaces: घरमा not घर मा)
- [ ] Using Devanagari danda (।) not Latin period (.)
- [ ] Compound verbs are one word (गर्नुहुन्छ not गर्नु हुन्छ)
- [ ] Vowel lengths are correct (पानी vs पनि, खुशी vs खुसि)
- [ ] Plural suffix uses long ू: -हरू not -हरु
- [ ] Adjectives agree with nouns (दिदी is feminine)
