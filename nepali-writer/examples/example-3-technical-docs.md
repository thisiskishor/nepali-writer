# Example 3: Technical Documentation

English → Nepali translation for software system requirements.

---

## English (Input)

```
System Requirements

The system requires administrator access to install updates. Users without 
administrator privileges will receive an error message. Minimum disk space 
required is 2GB. The application is compatible with Windows 10 and later.
```

---

## Nepali (Output)

```
प्रणाली आवश्यकताहरू

प्रणालीमा अपडेट स्थापना गर्न व्यवस्थापक पहुँच आवश्यक छ। व्यवस्थापक विशेषाधिकार नभएका
प्रयोगकर्ताहरूले त्रुटि सन्देश पाउनेछन्। न्यूनतम २ जीबी डिस्क स्थान आवश्यक छ।
यो अनुप्रयोग विन्डोज १० र पछिका संस्करणहरूसँग अनुकूल छ।
```

---

## Translation Notes

### Register & Tone
- **Register:** Semi-formal/Technical (instructional, clear, precise)
- **Tone:** Neutral, informational
- **Audience:** Technical users, system administrators
- **Context:** Help documentation or setup guide

### Key Translation Decisions

| English | Nepali | Why |
|---------|--------|-----|
| "System" | प्रणाली | Formal/technical; alternatives: सिस्टम (transliterated, also acceptable) |
| "Administrator access" | व्यवस्थापक पहुँच | पहुँच = access; व्यवस्थापक = administrator |
| "Update" | अपडेट | Transliterated (commonly used in tech); traditional: सुधार (less common in software context) |
| "Install" | स्थापना गर्न | Formal; literal: "place/establish" |
| "Privileges" | विशेषाधिकार | Formal/legal term for rights/permissions |
| "Error message" | त्रुटि सन्देश | त्रुटि = error, सन्देश = message |
| "Disk space" | डिस्क स्थान | Partially translated; डिस्क (transliterated), स्थान (Nepali for space/place) |
| "Application" | अनुप्रयोग | Formal Nepali; alternative: एप्लिकेसन (transliterated, also common) |
| "Compatible" | अनुकूल | Literally "suitable/fitting"; formal alternative: सामञ्जस्यपूर्ण |

### Terminology Reference Table

| Technical Term | Nepali (Formal) | Nepali (Alt.) | Transliterated |
|---|---|---|---|
| System | प्रणाली | तन्त्र | सिस्टम |
| Administrator | व्यवस्थापक | प्रबन्धक | एडमिनिस्ट्रेटर |
| Access | पहुँच | प्रवेश | एक्सेस |
| Update | अपडेट | सुधार | अपडेट |
| Install | स्थापना गर्न | राख्न | इन्स्टल गर्न |
| Privileges | विशेषाधिकार | अधिकार | प्रिभिलेज |
| Error | त्रुटि | गलती | एरर |
| Message | सन्देश | वार्ता | मेसेज |
| Disk space | डिस्क स्थान | डिस्क स्पेस | डिस्क स्पेस |
| Application | अनुप्रयोग | प्रोग्राम | एप्लिकेसन |
| Compatible | अनुकूल | सामञ्जस्यपूर्ण | कम्पेटिबल |

### Grammar Points Demonstrated

1. **SOV Structure:** Maintained even in technical sentences
2. **Postposition Attachment:** विषयमा, प्रणालीमा, हरूसँग (postpositions attached to nouns)
3. **Conditional/Causal:** नभएका (without) — negative participle
4. **Technical Clarity:** Formal register preserves precision
5. **Number Agreement:** २ जीबी (2 GB) — number + unit clarity

### Translation Strategy

In technical documentation, the mix of:
- **Formal Nepali terms** where available (प्रणाली, अनुप्रयोग)
- **Transliterated tech terms** where no standard Nepali equivalent exists (अपडेट, डिस्क)
- **Clear postposition attachment** for precision (स्थानमा not स्थान मा)

This approach serves both:
- Nepali speakers unfamiliar with English tech terms
- Technical users who expect certain standard terminology

---

## Disk Space Translation Note

⚠️ **Known Item for Review:** "Disk space" can be translated as:
- **डिस्क स्थान** (partially translated — recommended for formal Nepali)
- **डिस्क स्पेस** (fully transliterated — more common in tech communities)

Both are acceptable in Nepali technical documentation. Choose based on audience: formal docs → डिस्क स्थान; developer/tech-savvy audiences → डिस्क स्पेस.

---

## Common Mistakes to Avoid (in this context)

❌ **Wrong:** सिस्टमलाई अपडेट गर्न (incorrect postposition usage)
✓ **Correct:** प्रणालीमा अपडेट स्थापना गर्न (correct postposition -मा)

❌ **Wrong:** एडमिनिस्ट्रेटर पहुँच (mixing transliterated + Nepali)
✓ **Correct:** व्यवस्थापक पहुँच (consistent formal Nepali) OR एडमिनिस्ट्रेटर एक्सेस (consistent transliterated)

❌ **Wrong:** गलतीको सन्देश (wrong postposition)
✓ **Correct:** त्रुटि सन्देश (standard noun-noun compound)

❌ **Wrong:** यो एप्लिकेसन विन्डोज १० को साथ काम गर्छ (overly literal)
✓ **Correct:** यो अनुप्रयोग विन्डोज १० र पछिका संस्करणहरूसँग अनुकूल छ (proper tech register)

## When to Use This Translation Style

- Software documentation and help files
- System requirements and specifications
- Technical installation guides
- API documentation
- IT and infrastructure documentation
- User guides for technical products
